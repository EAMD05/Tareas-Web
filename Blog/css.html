CSS: Estilizando la Web

CSS (Cascading Style Sheets) es el lenguaje que utilizamos para dar estilo y diseño a nuestras páginas web. Aprende los conceptos fundamentales y cómo aplicarlos efectivamente.

¿Qué es CSS?

CSS es un lenguaje de hojas de estilo que se utiliza para describir la presentación de un documento HTML. Permite controlar el diseño, los colores, las fuentes y otros aspectos visuales de una página web. 

body {
    background-color: #f0f0f0;
    font-family: Arial, sans-serif;
}

Componentes Básicos de CSS

Selectores

Los selectores son patrones que se utilizan para seleccionar los elementos HTML a los que se aplicarán los estilos.

  • Selectores de tipo: p, h1, div
  • Selectores de clase: .clase
  • Selectores de ID: #id

Propiedades

Las propiedades son las características que queremos modificar en los elementos seleccionados.

  • color
  • font-size
  • margin
  • padding

Valores

Los valores son las especificaciones que asignamos a las propiedades.

  • Colores: #ff0000, red
  • Tamaños: 16px, 2em
  • Unidades: px, %, rem

Tipos de Selectores

Selectores por Tipo

Seleccionan elementos basados en su etiqueta HTML.

p {
    color: blue;
}

Selectores por Clase

Seleccionan elementos que tienen un atributo class específico.

.destacado {
    font-weight: bold;
}

Selectores por ID

Seleccionan un elemento único con un atributo id específico.

#encabezado {
    font-size: 24px;
}
