CSS: Estilizando la Web

CSS (Cascading Style Sheets) es el lenguaje que utilizamos para dar estilo y diseño a nuestras páginas web. Aprende los conceptos fundamentales y cómo aplicarlos efectivamente.

¿Qué es CSS?

CSS (Cascading Style Sheets o Hojas de Estilo en Cascada) es un lenguaje utilizado para describir la presentación de un documento HTML. Mientras HTML define la estructura del contenido (títulos, párrafos, listas, etc.), CSS se encarga de cómo se ve ese contenido (colores, tamaños, márgenes, posiciones, fuentes, etc.).

CSS permite separar el diseño del contenido, lo cual facilita:

  • El mantenimiento del código
  • La reutilización de estilos
  • Una mejor organización del código

Componentes Básicos de CSS

Selectores

Son la parte del CSS que indica a qué elementos HTML se le aplicarán los estilos.

p {
/* Estilos para todos los 
párrafos */
}

Propiedades

Son los aspectos visuales que se quieren modificar del elemento HTML.

color: blue;
font-size: 16px;
background-color: white;

Valores

Son los ajustes específicos que le das a las propiedades.

color: red; /* red 
es el valor */
font-size: 16px; /* 16px 
es el valor */

Tipos de Selectores

Selector por Tipo

Selecciona todos los elementos de un mismo tipo (etiqueta HTML).

h1 {
    color: green;
}

Afecta a todos los encabezados <h1> del documento.

Selector por Clase (.)

Selecciona todos los elementos que tienen un atributo class específico.

.titulo-principal {
    font-weight: bold;
}

Afecta a cualquier elemento con class="titulo-principal". Se puede reutilizar en varios elementos.

Selector por Identificador (#)

Selecciona un solo elemento con un atributo id específico.

#menu {
    background-color: gray;
}

Afecta únicamente al elemento que tenga id="menu". El id debe ser único en toda la página.

Ejemplo Completo

p {
    color: blue;
    font-size: 16px;
}

En este ejemplo:

  • p es el selector
  • color y font-size son propiedades
  • blue y 16px son los valores asignados
