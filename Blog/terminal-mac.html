Terminal de Mac: Guía Completa

Aprende los comandos más útiles de la Terminal de Mac para navegar, crear y gestionar archivos y directorios.

Navegación en la Terminal

La Terminal de Mac es una herramienta poderosa que te permite interactuar con tu sistema operativo a través de comandos. Aquí te presentamos los comandos más comunes y útiles.

Comandos de Navegación

  • pwd - Muestra el directorio actual
  • ls - Lista los archivos y carpetas en el directorio actual
  • cd [directorio] - Cambia al directorio especificado
  • cd .. - Sube un nivel en la jerarquía de directorios
  • cd ~ - Va al directorio home del usuario
  • clear - Limpia la pantalla de la terminal

Gestión de Archivos y Directorios

  • mkdir [nombre] - Crea un nuevo directorio
  • touch [nombre] - Crea un nuevo archivo vacío
  • rm [archivo] - Elimina un archivo
  • rm -r [directorio] - Elimina un directorio y su contenido
  • cp [origen] [destino] - Copia archivos o directorios
  • mv [origen] [destino] - Mueve o renombra archivos/directorios

Visualización de Contenido

  • cat [archivo] - Muestra el contenido de un archivo
  • less [archivo] - Muestra el contenido de un archivo con navegación
  • head [archivo] - Muestra las primeras líneas de un archivo
  • tail [archivo] - Muestra las últimas líneas de un archivo

Permisos y Propiedad

  • chmod [permisos] [archivo] - Cambia los permisos de un archivo
  • chown [usuario] [archivo] - Cambia el propietario de un archivo
  • ls -l - Muestra información detallada de archivos incluyendo permisos

Búsqueda

  • find [directorio] -name "[patrón]" - Busca archivos por nombre
  • grep "[texto]" [archivo] - Busca texto dentro de archivos
  • locate [archivo] - Busca archivos en todo el sistema

Procesos y Sistema

  • ps - Muestra los procesos en ejecución
  • top - Muestra información en tiempo real del sistema
  • kill [PID] - Termina un proceso
